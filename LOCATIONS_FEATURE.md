# הוספת לוקיישנים מותאמים אישית

## מה נוסף?

הוספתי אפשרות להוסיף לוקיישנים מותאמים אישית למאגר הלוקיישנים. 

### תכונות:
- ✅ כפתור "לוקיישן חדש" בעמוד הלוקיישנים
- ✅ מודאל להוספה ועריכה של לוקיישנים
- ✅ אפשרות לערוך ולמחוק לוקיישנים מותאמים (רק לוקיישנים שהוספת, לא את אלה המובנים)
- ✅ שמירה ב-Supabase
- ✅ סינון לפי אזור

### איך להשתמש?

1. **יצירת הטבלה ב-Supabase:**
   - היכנסי ל-Supabase Dashboard
   - לחצי על SQL Editor
   - העתיקי והדביקי את התוכן מקובץ `db/create_locations_table.sql`
   - הריצי את ה-SQL

2. **הוספת לוקיישן חדש:**
   - היכנסי לעמוד "לוקיישנים"
   - לחצי על "לוקיישן חדש"
   - מלאי את הפרטים: שם, אזור, סוג ותיאור
   - לחצי "שמירה"

3. **עריכה/מחיקה:**
   - לוקיישנים שהוספת יופיעו עם כפתורי "עריכה" ו"מחיקה"
   - הלוקיישנים המובנים (15 הראשונים) לא ניתנים למחיקה

### מבנה הטבלה:

```sql
locations:
  - id: מזהה ייחודי
  - title: שם הלוקיישן
  - region: אזור (center/north/south/jerusalem/sharon)
  - type: סוג (urban/nature/beach/village)
  - description: תיאור מפורט
  - created_at: תאריך יצירה
```

### קבצים ששונו:
1. `index.html` - נוסף מודאל ללוקיישן
2. `js/store.js` - נוספו פונקציות getLocations, saveLocation, deleteLocation
3. `js/ui.js` - עודכן renderLocations להציג כפתורים לעריכה/מחיקה
4. `js/app.js` - נוספו פונקציות openLocationModal, handleLocationSubmit, deleteLocation
5. `db/create_locations_table.sql` - SQL ליצירת הטבלה

---

**הערה:** אם הטבלה לא קיימת ב-Supabase, המערכת תציג רק את הלוקיישנים המובנים.
